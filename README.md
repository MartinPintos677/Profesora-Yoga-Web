# Samudra Vinyāsa — Landing Page

Landing page profesional para **Samudra Vinyāsa**, emprendimiento de Ashtanga Yoga enfocado en práctica consciente, respiración, presencia y movimiento.

El sitio fue desarrollado como una experiencia visual minimalista, espiritual y terapéutica, orientada a presentar la propuesta de clases, transmitir la identidad de marca y facilitar el contacto directo mediante WhatsApp e Instagram.

## 🌐 Demo

https://samudra-vinyasa.netlify.app/

## ✨ Objetivo del proyecto

El objetivo principal fue crear una página web moderna, responsive y visualmente coherente con la identidad de Samudra Vinyāsa.

La web busca:

- Presentar la marca y su concepto visual.
- Comunicar la propuesta de Ashtanga Yoga.
- Mostrar clases virtuales y presenciales.
- Presentar planes y precios.
- Facilitar la reserva de clases mediante WhatsApp.
- Transmitir una estética espiritual, terapéutica y profesional.

## 🧘 Sobre Samudra Vinyāsa

**Samudra Vinyāsa** significa “flujo del océano”.

La identidad visual del sitio está inspirada en:

- Movimiento.
- Respiración.
- Naturaleza.
- Calma.
- Transformación.
- Presencia.
- Conexión interior.

El sitio utiliza una estética oceánica y contemplativa, con predominio de tonos azules, fondos suaves, tipografía elegante y una composición visual orientada al bienestar.

## 🖼️ Secciones principales

La landing page incluye:

- **Hero principal** con identidad de marca.
- **La Práctica**, donde se explica el enfoque del Ashtanga Yoga.
- **Clases**, con información sobre modalidad virtual, presencial y práctica para todos los niveles.
- **Experiencia**, con una galería visual de la práctica.
- **Rafaela**, sección de presentación personal.
- **Planes**, con precios y opciones de práctica.
- **Contacto**, con acceso directo a WhatsApp e Instagram.
- **Footer**, con créditos de desarrollo.

## 🛠️ Tecnologías utilizadas

- Next.js
- React
- TypeScript
- Tailwind CSS
- Netlify

## 🤖 Uso de herramientas de IA

Para acelerar la etapa inicial de maquetación y exploración visual, se utilizó **v0.app** como herramienta de generación asistida de frontend.

Antes de utilizar v0.app se realizó una etapa previa de:

- Relevamiento de requerimientos.
- Definición de identidad visual.
- Dirección creativa.
- Estructura UX/UI.
- Selección de colores, estilo, tono y secciones.
- Definición de objetivos de conversión.

Luego del prototipo inicial generado con v0.app, el proyecto fue ajustado, personalizado y refinado manualmente, incluyendo:

- Correcciones visuales.
- Ajustes responsive.
- Integración de contenido real.
- Navegación entre secciones.
- Enlaces de contacto.
- Optimización de estructura.
- Deploy final en Netlify.

## 🎨 Dirección visual

El diseño fue pensado para evitar una estética genérica de fitness o gimnasio.

Se buscó una identidad más cercana a una marca boutique de bienestar:

- Minimalista.
- Espiritual.
- Terapéutica.
- Editorial.
- Suave.
- Orgánica.
- Premium.

La paleta visual se basa principalmente en:

- Azul oceánico profundo.
- Turquesa como acento.
- Tonos claros cálidos.
- Detalles suaves y naturales.

## 📱 Responsive Design

El sitio fue desarrollado con enfoque mobile-first, asegurando una correcta visualización en:

- Celulares.
- Tablets.
- Pantallas de escritorio.

Se priorizó especialmente la experiencia en mobile, considerando que gran parte del tráfico puede venir desde redes sociales.

## 📩 Contacto

El sitio permite contacto directo mediante:

- WhatsApp.
- Instagram.

La acción principal de conversión es la reserva o consulta de clases.
